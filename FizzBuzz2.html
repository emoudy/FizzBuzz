FizzBuzz!

The object of this game is to see if a range of numbers is evenly dividisble by 3 and 5.

If the number is evenly divisible by 3, you will see "Fizz".

If the number is evenly divisible by 5, you will see "Buzz".

If the number is evenly divisible by both 3 and 5, you will see "FizzBuzz".

If the number is NOT divisible by either, you will see the actual number.
